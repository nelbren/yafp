## :floppy_disk: Acquire

```bash
cd /usr/local
git clone https://github.com/nelbren/yafp.git
```

## :eyes: Preview

```bash
source /usr/local/yafp/yafp-ps1.bash
```

## :heavy_check_mark: Install

```bash
echo "source /usr/local/yafp/yafp-ps1.bash" | tee -a ~/.bashrc /etc/skel/.bashrc
```
